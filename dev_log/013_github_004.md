# Unit 13: GitHub Pages Migration - Subunit 4: Domain Migration

## Objective
Configure custom domain (allende.nz) for GitHub Pages deployment, update DNS records, and ensure proper HTTPS certificate provisioning.

## Implementation
- Create `CNAME` file in repository root with `allende.nz`
- Update DNS records at domain registrar:
  - Remove current A records pointing to AWS
  - Add CNAME record pointing to `<username>.github.io`
  - Configure apex domain if needed
- Test DNS propagation using `nslookup` and `dig`
- Verify HTTPS certificate auto-provisioning
- Test domain accessibility
